* This component is use in DashboardLayout.tsx
* To customize the code navigate to **/src/boilerplate/dashboard/components/Alerts.tsx** file  

The Alerts.tsx is a component that shows predefined information when an event happen.

## Create new Alarms Events 

* To config your alerts navigate to **/src/config/dashboard** and fill in the object 
with the require data


``` js

// ../src/config/dashboard
export const dashboardConfig = {
  links: {}
  alerts: {
    queryKey:"alert",
    actions:{
      // Customer role actions: 
      customer:[
        {
          queryValue: "payment_success",
          title: "Payment Successful üéâ",
          description: "You have successfully made your payment.üòÄ",
          variant: "success",
        },
        // rest of events
      ],
      // Admin role actions: 
      admin:[
        // admin events
      ],
    }
  }
}

```

## Trigger your event!

The Alerts.tsx component can only be displayed within the Dashboard layout, regardless of the route being displayed.
